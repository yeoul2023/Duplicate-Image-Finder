difPy Guide
===================================

**difPy** is a Python package that searches for images in one or more different folders and their sub folders, compares the images it found and checks whether these are duplicate or similar images.

Check out the :doc:`usage` section for further information, including how to :ref:`install` the project.

.. note::

   This project is under active development.
